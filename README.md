# Karaorust

Karaoke on console

## Deps

~~~
apt install libasound2-dev pkg-config
~~~

## Build

~~~
cargo build
~~~

## Run

~~~
cargo run --bin karaorust FILE
~~~

## Release

~~~
cargo build --release
~~~

## Tests

~~~
cargo test
~~~
